# Sachida's Artistry Studio

A luxury editorial portfolio website showcasing handcrafted resin keepsakes, varmala preservation, engagement platters, and wedding artistry by Sachida Sharma.

## Features

- **Modern Design**: Built with React and TanStack Router for smooth navigation
- **Responsive**: Fully responsive design that works on all devices
- **Type-Safe**: Written in TypeScript for better code quality
- **UI Components**: Powered by Radix UI and styled with Tailwind CSS
- **Fast**: Optimized with Vite for lightning-fast development and builds

## Tech Stack

- **Framework**: React 19 with TanStack Start
- **Routing**: TanStack Router
- **Styling**: Tailwind CSS 4
- **UI Components**: Radix UI
- **Forms**: React Hook Form with Zod validation
- **Build Tool**: Vite
- **Package Manager**: Bun

## Getting Started

### Prerequisites

- Node.js 18+ or Bun
- npm/yarn/pnpm/bun

### Installation

```bash
# Install dependencies
bun install

# Start development server
bun run dev

# Build for production
bun run build

# Preview production build
bun run preview
```

## Scripts

- `dev` - Start development server
- `build` - Build for production
- `build:dev` - Build in development mode
- `preview` - Preview production build
- `lint` - Run ESLint
- `format` - Format code with Prettier
